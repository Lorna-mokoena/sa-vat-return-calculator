# South African VAT Return Calculator

A simple and free web tool to help South African businesses and freelancers calculate VAT returns quickly and accurately.

## Features
- Calculate Output VAT (sales)
- Calculate Input VAT (expenses)
- VAT payable/refund calculator
- VAT threshold checker
- VAT201 help guide
- Simple and mobile-friendly design
